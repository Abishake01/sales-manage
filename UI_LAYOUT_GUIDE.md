## Responsive Behavior

```
Desktop (1200px+):
- Sidebar fixed 250px
- Content area responsive
- All columns visible
- Full functionality

Tablet (768px - 1200px):
- Sidebar fixed 200px
- Content area responsive
- Some columns may stack
- Touch-friendly buttons

Mobile (<768px):
- Sidebar reduced or collapsible
- Full-width content
- Horizontal scroll on tables
- Larger touch targets
- Stack layout for forms
```
